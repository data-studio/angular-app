## Getting started

1. Install [Vagrant](https://www.vagrantup.com/) by [HashiCorp](https://www.hashicorp.com/)
2. Clone/checkout/download this repository
3. Start the development VM with:
```shell
$ vagrant up
```
4. Direct your browser to `192.168.110.30`
